## Top 3 Insights for Aman

### 1. Technical Blocker Abandonment = Silent Dropoff Risk
**What Happened:**
- Ricardo's AI Trip Planner OpenAI integration never worked
- Quote: "For some reason, I would never get the freaking OpenAI results from the call"
- Had to choose: debug course project vs. focus on work transition
- **NEVER RESOLVED** - abandoned the project completely

**Why This Matters:**
- Students will deprioritize course when work conflicts arise
- No clear blocker resolution pathway = silent student dropoff
- Lost learning opportunity (debugging skills, troubleshooting)

**Recommended Fix:**
- **P0:** Implement Technical Blocker Triage System
  - #technical-help Slack channel (TA monitors daily)
  - "Known Issues" docs for each project (prevent Ricardo's issue for future students)
  - Weekly "Bring Your Blocker" office hours
  - 24-hour TA response SLA

---

### 2. Learner Personas: One-Size-Fits-All Support Doesn't Work
**Discovery:**
Ricardo represents a **different learner type** than Mike:

| Dimension | Mike (Guided Learner) | Ricardo (Autonomous Learner) |
|-----------|----------------------|------------------------------|
| **Background** | Can learn tech, needs structure | CS degree, prefers independence |
| **Setup Preference** | Benefits from walkthroughs | "Just give me all the steps" |
| **Pain Point** | Going too deep without guardrails | Live walkthroughs feel slow |
| **Support Style** | "Show me how" | "Let me do it myself" |
| **Value From TA** | Project scoping, accountability | Fast async support, pre-emptive docs |

**Why This Matters:**
- Current course serves one persona well but frustrates the other
- Ricardo: "It's hard to follow" when Aman does "click by click and bottom by bottom in front of me"
- Both types exist in every cohort - need dual approach

**Recommended Fix:**
- **P0:** Create Dual-Format Materials
  - "Quick Start Guide" (autonomous learners) - just the steps, no fluff
  - "Detailed Walkthrough" (guided learners) - step-by-step with screenshots
  - Let students choose their path

### 3. Setup Friction is UNIVERSAL (Confirmed 2/2 Students)
